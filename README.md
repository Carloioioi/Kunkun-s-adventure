# Kunkun's adventure

A simple Flappy Bird-liked game developed using Unity and C#.

## Features

- Player controls
- Obstacle spawning
- Collision detection
- Score system
- Game over screen

## Technologies

- Unity
- C#

## How to Play

Press **Space** to make Kunkun( the only Character you will see) jump.
Avoid the pipes and survive as long as possible.
